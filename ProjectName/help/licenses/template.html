License Name

You can put version info and disclaimers here, as well as information about the organization that made the license, which may not be the same as the licensor.

Please remember to update the <title> tag with your license name, too.

Section Name

You can give each section an id attribute to enable linking to sections of a license.

Any elements with a class name beginning with "param_" will be populated by URL query parameters.

For example, this text – 1776 – has the class "param_year." Linking to this license page with a query like template.html?year=2010 will set it to "2010." Give it a try!
